# Mini Consulting Projects

## 📌 概要
業務改善・課題発見・要件定義の練習用プロジェクト。
ビジネス課題を抽出し、解決案→要件定義→簡易プロトタイプを作成。

## 📁 構成
business_doc/ ※ 課題の深掘り
solution_design/ ※ 要件・解決案
prototype/ ※ 画面ラフ・モック


## 🎯 内容（例）
- 医療現場の残業削減
- データ加工の自動化提案
- 申請システムのペーパーレス化

## 💡 目的
- “考える力” を証明する  
- コンサル志望の転職で強力なアピールになる

## 📌 改善予定
- 業務フロー図追加
- モックアップを React で作成
