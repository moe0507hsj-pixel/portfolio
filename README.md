# Portfolio

このリポジトリは、バックエンド開発、データ加工、業務改善（コンサル的アプローチ）を総合的に示すためのポートフォリオです。  
実務に近い構成を意識し、技術力だけでなく「課題発見・要件定義・改善設計」までを一貫して学習・アウトプットしています。
「C# WebAPI × SQL × 要件定義」  
を軸にした実務想定のアウトプットを記録しています。

---
