# Data Processing（SQL / Python）

## 📌 概要
SQL Server を中心としたデータ加工プロジェクト。
実務を意識して「集計・加工・ETL」をテーマにしたサンプルを収録。

## 🛠 技術スタック
- SQL Server
- T-SQL
- Python（任意：加工スクリプト）
- pandas（任意）

## 📈 内容（サンプル）
- 売上データの集計
- 帳票用データ加工
- マスタとの結合・NULL 補完
- 日付テーブル作成
- データクレンジング

## 📁 ディレクトリ構成

sql/
python/
datasets/
README.md


## ▶️ 実行方法
- SQL Server / Azure Data Studio / SSMS で `sql/*.sql` を実行
- Python スクリプトは `python/` 配下で実行可能
